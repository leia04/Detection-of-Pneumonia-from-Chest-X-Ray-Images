# Dataset

This repository uses the pediatric chest X-ray dataset provided by Mendeley Data.

- Dataset: Labeled OCT and Chest X-Ray Images for Classification
- DOI: 10.17632/rscbjbr9sj.2
- URL: https://data.mendeley.com/datasets/rscbjbr9sj/2

## Note
Due to file size and licensing considerations, the image files are not included in this repository.

Please download the dataset from the original source and place it locally under the following structure:
data/
└── chest_xray/
├── train/
├── val/
└── test/
